## A custom New Tab page inspired by macOS Tahoe made with pure HTML and CSS.
The custom new tab page was made to replicate the aesthetics of macOS as much as possible, which includes:
# ✨ Features
- Dock with floating apps (shortcuts)
- Window containing NASA APOD and News embed.
- Search bar in the center with classic 'hello' placeholder (uses Borel font).
- Working glassy time, day and date on top.

# 📦 How to use

##### Chrome
1. Click the three dots in the top right and select Settings.
2. Click On startup from the left-hand menu.
3. Select Open a specific page or set of pages.
4. Click Add a new page, type the URL of the website you want, and click Add.
##### Edge
1. Click the three dots in the top right and select Settings.
2. Click Start, home, and new tabs from the left-hand menu.
3. Under When Microsoft Edge starts, select Open these pages.
4. Click Add a new page, enter your desired URL, and click Add.
